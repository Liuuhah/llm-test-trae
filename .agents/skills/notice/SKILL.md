# Notice Skill - 公司内部通知生成器

## 触发词
写通知、通知格式、起草通知、通知模板、部门通知、生成通知

## 核心规则

1. **标题格式**：必须以"**XX部通知**"开头（XX为具体部门名称）
2. **默认部门**：如果用户未提供部门信息，默认使用"XX部"
3. **部门署名**：在通知末尾添加部门署名，格式为"XX部"（不加粗）
4. **日期格式**：使用"YYYY年MM月DD日"格式
5. **语气风格**：正式、简洁、清晰

## 通知模板

```
**{部门}通知**

{通知正文内容}

特此通知。

{部门}
{日期}
```

## 示例

### 示例 1：默认部门
**用户输入**：帮我写一个五一劳动节放假的通知。

**输出**：
```
**XX部通知**

根据国家法定节假日安排，现将2026年五一劳动节放假事宜通知如下：

一、放假时间
2026年5月1日（星期五）至5月3日（星期日）放假调休，共3天。

二、注意事项
1. 请各部门提前做好工作安排，确保假期期间各项工作正常运转。
2. 放假期间请注意个人安全，遵守相关法律法规。

特此通知。

XX部
2026年4月27日
```

### 示例 2：指定部门
**用户输入**：我是销售部的，请帮我写一个五一劳动节放假的通知。

**输出**：
```
**销售部通知**

根据公司统一安排，现将销售部2026年五一劳动节放假事宜通知如下：

一、放假时间
2026年5月1日（星期五）至5月3日（星期日）放假调休，共3天。

二、工作要求
1. 请各位同事在放假前完成手头工作，确保客户业务不受影响。
2. 假期期间保持手机畅通，以便处理紧急事务。

特此通知。

销售部
2026年4月27日
```

## 执行步骤

1. 识别用户是否提供了部门信息
2. 如果没有提供，使用默认部门"XX部"
3. 根据用户需求生成通知正文
4. 按照模板格式组织内容
5. 添加当前日期
6. 输出完整通知
